  1. 视觉问答 / 医学视觉问答 （Vision Question Answering）

  1. (3D) 图像生成 （Text to Image）

开发框架

  • Gradio (重点关注)：
    • 一个用于快速创建机器学习界面的开源 Python 库。
    • Gradio 可以用于构建各种交互式界面，包括文本、图像、音频和视频输入的界面。它支持通过简单的 API 调用来添加输入和输出元素，无需复杂的前端代码。
  • Pytorch：
    • PyTorch（Python Torch）是一个开源的深度学习框架。
    • 它提供了一个灵活且动态的计算图，以及丰富的工具和库，使得深度学习任务更加便捷。
  • Django:
    • Django 是一个用于构建 Web 应用程序的开源 Python Web 框架。
    • Django 的目标是使开发者能够快速构建健壮、可维护且易于扩展的 Web 应用程序。因此，它通常被广泛用于需要迅速开发和部署的项目，如内容管理系统、博客平台、电子商务网站等。

几个案例

  • Video QA
  • Image(Medical) Caption
  • Visual(Medical) Question Answering
  • Text to Image(3D)

内容拓展思路

为了让论文的内容更充实，除了这些基本的算法实现之外建议朝以下两个方向做拓展：

  1. 增加业务内容

    搭建一个具有一定业务流程的系统，比如用户注册、登录、管理、具体算法的使用等；

  1. 模型优化

    着重描述算法模型，阐述算法模型结构、工作流程、创新点，或在你引用的模型之上增加自己的想法，对算法模型做优化；
